Marmitton sniffer

MS

Pourquoi utiliser cet outils ?

Si vous désirez ajouter une recette, marmitton sniffer vous permet d'ajouter les ingrédients d'une recette à votre liste de course

Comment utiliser cet outil ?

Les étapes :
  • Je glisse le bouton dans mes favoris
  • Je vais sur la page de recette marmitton
  • Je clique sur le bouton ajouté aux favoris
  • Je retrouve mes données dans une liste
